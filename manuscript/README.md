# Manuscript for PBDE, Thyroid, and Metabolic Health Study

This directory contains the LaTeX source and compiled PDF for the study: "Are serum levels of polybrominated diphenyl ethers (PBDEs) and other flame retardants associated with thyroid hormone levels and subsequent effects on lipid metabolism and insulin resistance?"

## Files
- `manuscript.tex`: Main LaTeX source file.
- `manuscript.pdf`: Compiled manuscript (APA style).
- `README.md`: This file.

## Compilation
The manuscript was compiled using `pdflatex` and `bibtex`. The figures are referenced from the `04-analysis/outputs/figures/` directory and tables from `04-analysis/outputs/tables/`.
